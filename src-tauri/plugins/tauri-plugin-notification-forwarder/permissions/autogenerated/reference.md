## Default Permission

Notification forwarding controls and status access.

#### This default permission set includes the following:

- `allow-check-permissions`
- `allow-get-permission-status`
- `allow-open-notification-listener-settings`
- `allow-drain-notifications`
- `allow-list-recent-notifications`
- `allow-wait-for-notifications`
- `allow-post-test-notification`
